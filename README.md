## Problem Overview

This is a **multilabel classification** problem where every image may contain **multiple attributes simultaneously**. The dataset also includes cases where a label is `"NA"` — meaning attribute information is missing.

The assignment required the following:
1. A deep-learning training script  
2. A loss curve plot (`training_loss` vs `iteration_number`)  
3. An inference script that prints predicted attributes for a given image  
4. Masking NA values correctly  
5. Handling label imbalance  
6. Using pretrained architectures (ResNet / Inception, etc.)  
7. Clean modular code  

All requirements have been implemented.

## Handling Missing Labels (NA)

The dataset contains entries like:

```
image_5.jpg   NA  0  1  NA
```

We cannot drop these samples because each image still contributes to the learning of other attributes.

Solution implemented:
- Convert `"NA"` to a **dummy label** → `0`
- Create a **mask vector** → `[0/1 per attribute]`  
- Compute BCE loss only on attributes where mask = 1

This ensures:
- No data loss  
- No false learning from NA values  

---

## Handling Class Imbalance

Multilabel problems often have uneven attribute distribution.  
We compute **positive weights** per attribute using:

```
pos_weight = (num_negative_samples / num_positive_samples)
```

These weights are used during training to help the model learn **rare attributes** more effectively.

## Model Architecture

- Backbone: **ResNet50 (ImageNet-pretrained)**
- Final layer replaced with:

```
Linear(2048 → 4)
Sigmoid()     # independent probabilities for each attribute
```

Why Sigmoid?  
Because each attribute is independent (typical multilabel setup).
